
# RegAlign AI
AI-powered Regulations and Compliance Assistant for Saudi Organizations

## Overview
RegAlign AI is an AI-powered compliance assistant designed to simplify cybersecurity compliance for Saudi organizations, particularly in regulated sectors like finance, energy, and government. It automates asset-based mapping to NCA and SAMA controls, reducing manual effort, improving accuracy, and supporting audit readiness.

## Features
- **Asset-Based Compliance Mapping**: Enter asset details (type, sector, data classification, hosting) and receive recommended NCA/SAMA controls.
- **AI-Powered Chatbot**: Ask regulation-specific questions and receive structured, regulation-grounded responses.
- **Report Generation**: Export compliance guidance and evidence suggestions as PDF.
- **Retrieval-Augmented Generation (RAG)**: Ensures all guidance is grounded in authoritative documents.

## Technology Stack
- **Google AI Studio** for development
- **Gemini LLM** as the language model
- **RAG (Retrieval-Augmented Generation)** for regulation-grounded responses
- **PDF Export** functionality for audit documentation

## Methodology
- Prototype-driven development
- Structured asset input forms for precise control mapping
- AI-generated structured reports
- Custom prompt engineering for improved accuracy

## Evaluation
- **Speed Improvement**: ~89% faster than manual compliance mapping
- **Accuracy Improvement**: Expected ~85.98% more accurate than manual process
- Three use cases demonstrate consistent efficiency and correctness improvements.
